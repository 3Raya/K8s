# K8s provisoning using kubeadm


#### Server requirements

[K8s server requirements](https://kubernetes.io/docs/setup/production-environment/tools/kubeadm/install-kubeadm "k8s server requirements")

| NODE   | OS                                | CPU | RAM |
| ------ | --------------------------------- | --- | --- |
| MASTER | Ubuntu 24.04.1 LTS (Noble Numbat) | 2   | 8   |
| WORKER | Ubuntu 24.04.1 LTS (Noble Numbat) | 2   | 4   |


#### Execute the script

1. Download **master-node.sh** on master node and run with sudo
2. Download **worker-node.sh** on master node and run with sudo
